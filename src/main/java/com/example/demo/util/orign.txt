如果您想更改右上角的 Bing 按钮，有两种方法可以尝试：

方法一：（不可用）
在 Edge 浏览器的设置中，找到「搜索」选项，然后关闭「显示 Bing 聊天按钮」。¹
方法二：（未测试）
在注册表编辑器中，导航到 HKEY_CURRENT_USER\Software\Microsoft\Edge\Main ，然后创建一个名为 HideBingChatButton 的 DWORD 值，并将其设置为 1 。

方法三：（可以用）
使用命令参数隐藏 Bing 按钮。您需要在 Microsoft Edge 的快捷方式属性中添加 –disable-features=msUndersideButton 字符串。²
方法四：（未测试）
进入设置 > 侧边栏 >“发现”，并关闭“显示发现”选项23。这个选项只在 ​Edge Canar​​ 113.0.1742.0 或更高版本中可用。
